# myself-security（当前版本1.0.0）

## 简介
基于Spring Security + Spring Social + Spring Security OAuth + REST服务开发的案例教程，Maven多模块开发，根据推文更新项目。
 
## 特点
同步推文，文章讲解，注释详细，信息具体，版本最新

## 功能

    浏览器（Session方式）、App（Token方式）
    用户密码登录
    第三方登录
    手机短信登录
    配置静态权限
    依赖数据库服务提供动态权限
    集群session管理
    SSO单点登录
